zklighter - creates JavaScript and CSS files for ZK Light
---------

Steps to Crate JavaScript and CSS files for ZK Light

1. ./build clean zk zul zklighter;./build zk zul zklighter

2. zklighter/build

3. Check in zkligher/lib

4. Merge SVN to ZK Light project
	(https://zk1.svn.sourceforge.net/svnroot/zk1/branches/5.0/zklighter/lib)
